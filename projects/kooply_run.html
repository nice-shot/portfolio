Kooply Run

Kooply is a mobile gaming startup that specializes on games with built-in editors that allow players to create and share their own levels. Kooply Run is the company's flagship app. It's a 3D runner game in the vein of Subway Surfers and Sonic Dash and it allows players to create unique levels on their phones using a very powerful editor.

Been working at the company since 2023 as a Unity Developer as part of the level creators squad. Introduced features like different level types, music library, community level competitions, inbox, comments and many more.
